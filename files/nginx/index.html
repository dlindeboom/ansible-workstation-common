Hey there fellow developer,

Looks like you're running into some trouble huh? ( ͡° ͜ʖ ͡°).

First of all check your url link.

If you are setting up a new site, continue reading.

Just create a new file in /home/USERNAME/nginx.

In that file use the code below to configure your host.

server {
    listen 80;
    server_name www.example.com.localhost;
    root /home/USERNAME/Projects/PATH;
    index index.php;

    include default.conf;
    include wordpress.conf; # Only for Wordpress
    include magento.conf; # Only for Magento 1
    include magento2.conf; # Only for Magento 2

    location ~ \.php$ {
        fastcgi_pass php72; # Available PHP versions: php72, php73, php74
        include php.conf;
    }
}
